# Ntando Masangu's Full Stack Porfolio Website

Personal website portfolio built using React.JS, Vite and CSS Modules and Flexbox.

# Live Demo

https://ntando-masangu-portfolio.vercel.app/

# Features

- Responsive design that is mobile friendly
- Light and Dark mode toggle
- Smooth scrolling navigation throughout the website
- Active and fixed navigation section and highlighting
- Contact Form with real email delivery using EmailJS (https://www.emailjs.com/)
- Modern gradient styling and hover effects
- Creative, clean, modern and accessible layout

# Tech Stack

- React.JS (Vite)
- CSS Flexbox and Modules
- Ionicons for the custom icons
- EmailJS for the contact form
- Intersection observer

# Setup and Local Run

- Clone repo:
  ```bash
  git clone https://github.com/ntando-masangu/FUTURE_FS_01
  cd FUTURE_FS_01
  ```
- Install dependencies
  npm install

- Create env file with EmailJS credentials
  VITE_EMAILJS_SERVICE_ID= ""
  VITE_EMAILJS_TEMPLATE_ID=""
  VITE_EMAILJS_PUBLIC_KEY=""

- Start the dev server
  npm run dev
